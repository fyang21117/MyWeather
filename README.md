Cool the weather is a weather forecast software based on open source Android end, with the view of the provinces county, query any city weather, free switch, manual update automatically updates the weather weather, the background, and other functions. Weather data in cool weather is provided by mild weather, background images are provided by bing, and the code follows the Apache v2 License open source protocol. This software is mainly used for learning and communication.
